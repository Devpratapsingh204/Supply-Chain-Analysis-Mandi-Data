# 📦 Supply Chain Analysis using Government Data

## 📌 Project Overview
This project analyzes mandi price data from the Government of India to understand price variations, market instability, and demand-supply imbalance.

---

## 📊 Dataset
- Source: data.gov.in  
- Records: 7000+  
- Fields: State, Market, Commodity, Prices  

---

## ⚙️ Features
- Data Cleaning and Preprocessing  
- Feature Engineering (price_spread, market_pressure)  
- Risk Classification  
- State-wise and Commodity-wise Analysis  
- Interactive User Input  

---

## 📈 Key Insights
- Prices vary across states indicating demand-supply imbalance  
- Some markets show unstable pricing patterns  
- High market pressure indicates higher demand  

---

## 🛠 Tools Used
- Python  
- Pandas  
- Matplotlib
